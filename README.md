<h2 id="changing-the-title-and-description" tabindex="-1"><a class="heading-link" href="#changing-the-title-and-description">Changing the title and description<span class="heading-link-symbol" aria-hidden="true"></span></a></h2>
<p>By default, the title of your site is <code>username.github.io</code>. You can change the title by editing the <code>_config.yml</code> file in your repository. You can also add a description for your site.</p>
<ol>
<li>
<p>Click the <strong>Code</strong> tab of your repository.</p>
</li>
<li>
<p>In the file list, click <code>_config.yml</code> to open the file.</p>
</li>
<li>
<p>Click <svg version="1.1" width="16" height="16" viewBox="0 0 16 16" class="octicon octicon-pencil" aria-label="The edit icon" role="img"><path d="M11.013 1.427a1.75 1.75 0 0 1 2.474 0l1.086 1.086a1.75 1.75 0 0 1 0 2.474l-8.61 8.61c-.21.21-.47.364-.756.445l-3.251.93a.75.75 0 0 1-.927-.928l.929-3.25c.081-.286.235-.547.445-.758l8.61-8.61Zm.176 4.823L9.75 4.81l-6.286 6.287a.253.253 0 0 0-.064.108l-.558 1.953 1.953-.558a.253.253 0 0 0 .108-.064Zm1.238-3.763a.25.25 0 0 0-.354 0L10.811 3.75l1.439 1.44 1.263-1.263a.25.25 0 0 0 0-.354Z"></path></svg> to edit the file.</p>
</li>
<li>
<p>The <code>_config.yml</code> file already contains a line that specifies the theme for your site. Add a new line with <code>title:</code> followed by the title you want. Add a new line with <code>description:</code> followed by the description you want. For example:</p>
<pre><code class="hljs language-yaml"><span class="hljs-attr">theme:</span> <span class="hljs-string">jekyll-theme-minimal</span>

</code></pre>
</li>
<li>
<p>When you are finished editing the file, click <strong>Commit changes</strong>.</p>
